DevOps Engineer

Location: California

Job Overview

A Devops engineer combines the understanding of infrastructure provisioning, application packaging, deployment and programming. The goal is to build automation around the following tasks: infrastructure provisioning in the cloud, application packaging into VMs or docker containers, application deployment, CI/CD, monitoring, logging and alerting. We expect the candidate to have done these tasks before in one of their previous roles. At Kami Vision, we want to leverage latest cloud technology in public clouds, infrastructure as code and server automation using programming in languages like terraform, python, ansible. One of the key goals of a devops engineer is to improve the overall productivity of other developers and application release process.


Responsibilities

  • Deploy infrastructure in one or more of the public clouds like AWS, GCP or Azure
  • Automate application packaging using docker containers
  • Deploy application using a set of containers and integrate with other cloud services used in the application
  • Automate deployment and monitoring using technologies like Kubernetes, APM tools and logging
  • Build a CI/CD pipeline for automated application testing as new checkins are done. This could be using tools like Jenkins.
  • Set up alerts for production deployment and integrate with one or more alerting tools
  • Work with developers to improve their productivity in writing code, testing and checkins
  • Work with SRE team to automate troubleshooting of various systems


Skills and Qualification

  • Good understanding of cloud technologies like AWS, VMs, VPCs, security groups, S3, RDS
  • Experience in writing automation code using python, terraform and ansible
  • Knowledge of building monitoring and dashboards for an application using tools like nagios, prometheus etc.
  • Experience in setting up logging and alerting using ELK
  • Knowledge of best practices around cloud infrastructure deployment, security and compliance
  • Willingness to learn new tools, technologies and continuous improvement
  • Good team player who can work with developers, SREs and customer support teams
  • Bachelor's or Master's degree in computer science or software engineering is preferred
  • Prior work experience in doing automation on cloud is highly desired
  • Experience with docker containers, kubernetes or equivalent systems is required
